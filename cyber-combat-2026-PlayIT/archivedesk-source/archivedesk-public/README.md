# ArchiveDesk

ArchiveDesk is an evidence-archive review console (HTTP): register and login,
create cases, upload ZIP evidence bundles, run the analyzer, and view reports,
with separate user and admin workflows.

- Public port: `8130`
- Flag: `/flag.txt`
- Proof: `/proof.txt`
- State: `/opt/ad/state`

Patch the service while keeping the normal user and admin workflows working for
the checker.
